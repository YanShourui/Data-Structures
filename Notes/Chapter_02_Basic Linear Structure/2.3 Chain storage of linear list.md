### 单链表
1.结点类型
```
typedf struct lnode
{
  datatype data;
  struct lnode *next;//next是一个指向struct lnode 类型的指针。
}LNode,*LinkLlist;//LNode是Link List Node的缩写；*LinkLlist是指向LNode的指针。
```
2.定义链表
```
LinkLlist H;
```
3.定义结点指针
```
LNode *p;
```
